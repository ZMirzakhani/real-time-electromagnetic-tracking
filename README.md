# Real-Time Electromagnetic Tracking System

Real-time electromagnetic tracking (EMT) system for 3D localization and orientation estimation using embedded sensing, signal processing, and state estimation techniques.

---

## Overview

This project presents the design and implementation of a real-time electromagnetic tracking system developed for accurate 3D position and orientation estimation.

The system integrates custom transmitter/receiver hardware, embedded acquisition modules, signal conditioning circuits, and MATLAB-based processing algorithms to reconstruct trajectories under noisy measurement conditions.

Experimental evaluations demonstrated stable real-time tracking performance with sub-centimeter positioning accuracy.

---

## Features

- Real-time electromagnetic tracking
- 3D localization and orientation estimation
- Embedded data acquisition
- Signal filtering and calibration
- MATLAB-based visualization
- Noise reduction and state estimation
- Experimental validation in 2D and 3D environments
